# A simple NodeJS repository to clone.

## Packages:
    - express
    - ejs
    - body-parser
    - nodemon
  
## How to clone it ?
    - git clone https://github.com/ayranlouro/nodejs-repo.git

## How to run server ?
    - npm install
    - npm start
